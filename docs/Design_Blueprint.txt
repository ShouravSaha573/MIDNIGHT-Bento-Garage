Sunrise Coral Glass — Complete Design & Development Blueprint

এই document অনুযায়ী “Midnight Bento Garage — Sunrise Coral Glass” design ও project build করলে assignment-এর functionality, structure, visual consistency এবং development planning একসঙ্গে maintain করা যাবে।

Assignment-এর core requirement হলো: client appointment form পূরণ করবে, নির্দিষ্ট তারিখে available mechanic নির্বাচন করবে, প্রতি mechanic প্রতিদিন সর্বোচ্চ ৪টি appointment নিতে পারবে, একই client একই দিনে duplicate appointment নিতে পারবে না এবং Admin appointment list দেখে date ও mechanic পরিবর্তন করতে পারবে। Projectটি HTML, CSS, JavaScript, PHP ও MySQL দিয়ে তৈরি এবং online server-এ চালু থাকতে হবে।

23. Duplicate Appointment Rule

Assignment বলছে একই client নির্দিষ্ট তারিখে আগে appointment নিয়ে থাকলে আবার appointment নিতে পারবে না।

Client identify করার practical rule:
Phone Number + Appointment Date

Database check:

SELECT COUNT(*)
FROM appointments
WHERE phone = ?
AND appointment_date = ?;

Result ১ বা তার বেশি হলে:
- Reject appointment
- Show duplicate warning


24. Mechanic Capacity Rule

প্রতি mechanic প্রতিদিন maximum ৪টি appointment নিতে পারবে।

SELECT COUNT(*)
FROM appointments
WHERE mechanic_id = ?
AND appointment_date = ?;

Logic:

Count < 4
→ Appointment allowed

Count = 4
→ Mechanic fully booked
→ Appointment rejected

Frontend display database count-এর সঙ্গে match করতে হবে।

Toast:

- Heavy parallax
- Excessive moving background

37. Database Structure

Mechanics table:

CREATE TABLE mechanics (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL
);

Appointments table:

CREATE TABLE appointments (
    id INT AUTO_INCREMENT PRIMARY KEY,
    client_name VARCHAR(100) NOT NULL,
    address VARCHAR(255) NOT NULL,
    phone VARCHAR(30) NOT NULL,
    car_license VARCHAR(50) NOT NULL,
    car_engine VARCHAR(50) NOT NULL,
    appointment_date DATE NOT NULL,
    mechanic_id INT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (mechanic_id) REFERENCES mechanics(id)
);

38. Security and Error Handling

PHP database:
- PDO বা MySQLi prepared statement ব্যবহার করতে হবে।

- User input sanitize করতে হবে।
- Server-side validation mandatory।
- Database error directly user-কে দেখানো যাবে না।
